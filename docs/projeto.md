# Projeto - Sistema de Vendas

## Objetivo
Construir um sistema de vendas com foco em organização, desempenho e futura integração com Power BI.

## Tecnologias
- Next.js
- TypeScript
- Supabase
- CSS / Tailwind (se adotado)

## Escopo atual
- Cadastro de vendas
- Cadastro de itens da venda
- Resumo por venda
- Consulta de vendas
- Estrutura preparada para relatórios

## Escopo futuro
- Dashboard interno
- Integração com Power BI
- Controle de usuários
- Relatórios por cliente, produto e período

## Princípios do projeto
- Código limpo
- Componentização
- Regras de negócio documentadas
- Estrutura pronta para crescer sem bagunça
