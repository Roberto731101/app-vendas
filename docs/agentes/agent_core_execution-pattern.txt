# Padrão de Execução

Sempre seguir esta sequência:

## 1. Ler
- Identificar arquivos envolvidos
- Identificar domínio

## 2. Diagnosticar
- O que está errado?
- O que está faltando?

## 3. Planejar
- Listar alterações EXATAS
- Não executar ainda

## 4. Executar
- Alterar apenas arquivos do escopo
- Nunca tocar fora do domínio

## 5. Validar
- Conferir consistência
- Conferir permissões
- Conferir UI

## 6. Retornar
- Resumo objetivo
- Arquivos alterados
- Próximo passo sugerido
